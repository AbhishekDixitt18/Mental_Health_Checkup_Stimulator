# Mental Health Companion 💚

A beautiful, interactive web application for tracking your mental health and well-being.

## Features ✨

- **Mood Tracking**: Record your emotional state with 5 different mood levels
- **Personalized Support**: Get inspiring quotes and exercises based on your mood
- **History Tracking**: View your complete mood journey over time
- **Statistics**: Analyze your mood patterns with beautiful visualizations
- **Wellness Resources**: Access calming exercises and motivational quotes
- **Responsive Design**: Works beautifully on desktop, tablet, and mobile devices

## How to Run 🚀

### Option 1: Using Python (Recommended)

1. Make sure Python is installed on your system
2. Open a terminal in this directory
3. Run the server:
   ```bash
   python server.py
   ```
4. The application will automatically open in your browser at `http://localhost:8000`

### Option 2: Using Node.js

If you have Node.js installed, you can use:
```bash
npx http-server -p 8000 -o
```

### Option 3: Using VS Code Live Server

1. Install the "Live Server" extension in VS Code
2. Right-click on `index.html`
3. Select "Open with Live Server"

## Usage Guide 📖

1. **First Time Setup**
   - Enter your name and age
   - Click "Begin Your Journey"

2. **Track Your Mood**
   - Click "Track Your Mood" on the dashboard
   - Select how you're feeling (Very Sad to Very Happy)
   - Optionally add notes about your feelings
   - Save your entry

3. **Get Support**
   - After saving a mood entry, receive personalized support
   - Read inspiring quotes
   - Try recommended calming exercises

4. **View History**
   - See all your previous mood entries
   - Review notes and timestamps

5. **Check Statistics**
   - View mood distribution
   - See percentages and patterns
   - Track your progress over time
